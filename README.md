# 📿 أثَر | ATHAR

**رفيقك اليومي في عالم العبادة والمعرفة**

بوت تيليغرام شيعي شامل يوفر مكتبة دينية متكاملة مع نظام إشعارات يومية ذكي، مبني بـ Python + aiogram.

| الميزة | الوصف |
|--------|-------|
| 📿 **الأدعية** | إرسال ملفات PDF للأدعية مباشرة من قناة تيليغرام عبر `file_id` |

## 🕌 خوارزمية مواقيت الصلاة

يعتمد البوت على **المذهب الجعفري (الاثنا عشري)** مع تصحيح المنطقة الزمنية عبر `pytz`:

| الصلاة | الزاوية / الطريقة |
|--------|-----------------|
| الفجر | -16° (معيار العراق الجعفري) |
| الشروق | -0.833° (انكسار ضوئي) |
| الظهر | الزوال الحقيقي |
| العصر | ظل الشيء مرتين |
| المغرب | -4° بعد الغروب |
| العشاء | -14° |
| منتصف الليل | منتصف المسافة بين المغرب والفجر |

> جميع التوقيتات مرتبطة بـ `config.TIMEZONE` (افتراضياً `Asia/Baghdad`) عبر `pytz` لضمان الدقة بغض النظر عن توقيت السيرفر.

## 📚 هيكل ملفات JSON

### المحتوى النصي (hadith, wisdom)
```json
{
  "items": [
    {
      "id": "H001",
      "text": "نص الحديث",
      "author": "الإمام الصادق عليه السلام",
      "source": "الكافي"
    }
  ]
}
```

### الأدعية بصيغة PDF
```json
{
  "items": [
    {
      "id": "D001",
      "title": "دعاء كميل",
      "is_pdf": true,
      "file_id": "BQACAgIA..."
    }
  ]
}
```

> البوت يفحص حقل `is_pdf` تلقائياً: إن كان `true` يُرسل الملف عبر `answer_document`، وإن كان النص موجوداً يُرسله كرسالة نصية.

### تعقيبات الصلاة
```json
{
  "metadata": { "prayer": "الفجر", "total": 37 },
  "items": [
    {
      "id": "F001",
      "title": "تعقيب الفجر",
      "text": "نص التعقيب"
    }
  ]
}
```

> التعقيبات تُعرض بحد 500 حرف للعنصر الواحد مع عداد للباقي، لتجنب تجاوز حد تيليغرام (4096 حرف).

### المناسبات الهجرية
```json
{
  "items": [
    {
      "hijri_date": "10-01",
      "text": "اليوم العاشر من محرم...",
      "amal": "الأعمال المستحبة"
    }
  ]
}
```

> صيغة `hijri_date` هي `"DD-MM"` (يوم-شهر). يدعم البوت مناسبات متعددة في نفس اليوم.

## 🛡️ ملاحظات تقنية

- **قاعدة البيانات:** SQLite محلية
- **التوقيت:** جميع الحلقات تستخدم `pytz` مع `config.TIMEZONE` — لا تأثير لتوقيت السيرفر
- **الأدعية PDF:** ترسل عبر `file_id` مباشرة من سيرفرات تيليغرام دون إعادة رفع
- **HTML escaping:** كل النصوص القادمة من JSON تمر بـ `_escape_html()` لمنع خطأ `CantParseEntities`
- **عدم تكرار المحتوى:** يتتبع البوت ما أُرسل لكل مستخدم يومياً ويُعيد التعيين منتصف الليل
- **pagination:** حد 64 بايت لـ `callback_data` محمي في `navigation_service.py`
- **back buttons:** كل زر رجوع مُرتبط بالـ callback الصحيح المسجل في `register_handlers`
