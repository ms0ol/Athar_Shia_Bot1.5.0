# 📿 أثَر | ATHAR

**مرافقك اليومي في عالم العبادة والمعرفة**

بوت تيليغرام شيعي شامل يوفر مكتبة دينية متكاملة مع نظام إشعارات يومية ذكي.

---

## ✨ المميزات

| الميزة | الوصف |
|--------|-------|
| 📿 **الأدعية والزيارات** | مكتبة كاملة من الأدعية المأثورة والزيارات والمناجيات بصيغة PDF |
| 🕌 **مواقيت الصلاة** | حساب دقيق وفق المذهب الجعفري (الفجر -16°، المغرب -4°) |
| 📿 **التعقيبات** | تعقيبات صلوات الفجر والظهر والمغرب والعشاء |
| 🗓 **المناسبات الهجرية** | تنبيهات يومية بالمناسبات والأعمال الدينية |
| 📖 **المحتوى اليومي** | حديث يومي، حكمة، دعاء، مناجاة — بنظام عدم التكرار |
| 🔔 **اشتراكات ذكية** | إشعارات مخصصة لكل مستخدم (صلاة، محتوى، مناسبات) |
| 🌍 **دعم المدن** | مواقيت لأكثر من 30 مدينة عربية وإيرانية |

---

## 🚀 التشغيل السريع

### المتطلبات

```
Python 3.9+
aiogram 2.x
pytz
hijri_converter
```

### التثبيت

```bash
pip install aiogram pytz hijri-converter
```

### إعداد متغيرات البيئة

```bash
export BOT_TOKEN="your_telegram_bot_token"
export ADMIN_IDS="123456789"          # معرّف المشرف
export CHANNEL_ID="@your_channel"     # قناة تيليغرام للبوت
export TIMEZONE="Asia/Baghdad"        # المنطقة الزمنية
export CITY="بغداد"                   # المدينة الافتراضية
export LATITUDE="33.3152"
export LONGITUDE="44.3661"
```

### التشغيل

```bash
python app.py
```

## 📁 هيكل المشروع

```
athar-bot/
├── app.py                      # نقطة التشغيل الرئيسية
├── config.py                   # الإعدادات ومتغيرات البيئة
├── database.py                 # قاعدة بيانات SQLite

│
├── services/
│   ├── content_service.py      # خدمة المحتوى الديني
│   ├── event_service.py        # خدمة المناسبات الهجرية
│   ├── navigation_service.py   # بناء لوحات المفاتيح
│   ├── prayer_service.py       # حساب مواقيت الصلاة
│   └── subscription_service.py # إدارة اشتراكات المستخدمين
│
└── data/
    └── normalized/
        ├── daily_content/      # محتوى يومي (حديث، حكمة، دعاء)
        ├── event_content/      # المناسبات الهجرية والأدعية الأسبوعية
        ├── library/            # مكتبة الزيارات والمناجيات
        └── prayer_content/     # تعقيبات الصلوات الخمس
```

## 🕌 خوارزمية مواقيت الصلاة

يعتمد البوت على **المذهب الجعفري (الاثنا عشري)**:

| الصلاة | الزاوية / الطريقة |
|--------|-----------------|
| الفجر | -16° (معيار العراق) |
| الشروق | -0.833° (انكسار ضوئي) |
| الظهر | الزوال الحقيقي |
| العصر | ظل الشيء مرتين |
| المغرب | -4° بعد الغروب |
| العشاء | -14° |
| منتصف الليل | منتصف المغرب والفجر |

## 📚 هيكل ملفات JSON

### المحتوى اليومي (hadith, wisdom, daily_dua)
```json
{
  "items": [
    {
      "id": "H001",
      "text": "نص الحديث",
      "author": "الإمام الصادق عليه السلام",
      "source": "الكافي"
    }
  ]
}
```

### الأدعية بصيغة PDF
```json
{
  "items": [
    {
      "id": "D001",
      "title": "دعاء كميل",
      "is_pdf": true,
      "file_id": "BQACAgIA..."
    }
  ]
}
```

### تعقيبات الصلاة
```json
{
  "metadata": { "prayer": "الفجر" },
  "items": [
    {
      "id": "F001",
      "title": "تعقيب الفجر",
      "text": "نص التعقيب"
    }
  ]
}
```

### المناسبات الهجرية
```json
{
  "items": [
    {
      "hijri_date": "01-01",
      "text": "أول محرم...",
      "amal": "الأعمال المستحبة"
    }
  ]
}
```

## 📝 ملاحظات تقنية

- قاعدة البيانات: **SQLite** محلية عبر `database.py`
- جميع التوقيتات مرتبطة بمنطقة `Asia/Baghdad` عبر **pytz**
- الأدعية المرفوعة على تيليغرام ترسل عبر `file_id` مباشرةً دون إعادة رفع
- نظام عدم التكرار: يتتبع البوت المحتوى المرسل لكل مستخدم يومياً
- التعقيبات: تُعرض تدريجياً مع حماية من تجاوز حد تيليغرام (4096 حرف)
